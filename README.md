To develop locally:

```
npm install
vc dev
```

```
open http://localhost:3000
```

API Documentation:

The API documentation is available via Swagger UI at:

- Development: http://localhost:3000/api-docs
- Production: https://matti-assets-api.vercel.app/api-docs

Postman Collection:

Import the complete API collection into Postman for easy testing:

- Direct import: https://www.postman.com/collections/import-from-url?url=https://matti-assets-api.vercel.app/postman-collection.json
- Download JSON: https://matti-assets-api.vercel.app/postman-collection.json

To build locally:

```
npm install
vc build
```

To deploy:

```
npm install
vc deploy
```
