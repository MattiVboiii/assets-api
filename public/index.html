Welcome to the Assets API! This API provides information about various assets including cars, boats, houses, and planes with their prices formatted in European style.

Note: All prices are displayed with European number formatting (using periods as thousands separators, e.g., 10.000 instead of 10000).
Notice: I took some time making this API, it's something simple I wanted to re-use for myself, but I thought "Why not make it public?", but high chance will be that I won't update it, unless something is broken.
Security: This API includes rate limiting (100 requests per 15 minutes), CORS protection, security headers, and input validation to prevent abuse and ensure safe usage.

Query Parameters

You can use query parameters to sort, filter, and limit the results from list endpoints:

Sorting

  • sort=price - Sort by price
  • sort=name - Sort by name (alphabetical)
  • sort=id - Sort by ID
  • order=asc - Sort in ascending order (default)
  • order=desc - Sort in descending order

Searching

  • search=toyota - Search assets by name (case-insensitive)

Filtering

  • min_price=50000 - Show only assets with price >= 50,000
  • max_price=100000 - Show only assets with price <= 100,000

Limiting

  • limit=5 - Limit results to first 5 items
Examples:
GET /cars?sort=price&order=asc - Cars sorted by price (lowest first)
GET /boats?min_price=100000&max_price=500000 - Boats between 100k-500k
GET /houses?sort=name&limit=3 - First 3 houses alphabetically
GET /planes?search=cessna - Planes with "cessna" in the name
GET /assets?sort=price&order=desc&limit=10 - Top 10 most expensive assets from all categories

Response Format

Each endpoint returns an array of asset objects with the following structure:

{
  "id": number,
  "name": string,
  "price": string, // Formatted with European thousands separator
  "picture": string, // URL to asset image
  "type": string // Only included in /assets endpoint ("car", "boat", "house", "plane")
}
